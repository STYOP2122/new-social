# New Social — React + Firebase

Социальное мини‑приложение на **React + TypeScript** с авторизацией и базовыми соц‑фичами (лента, друзья, уведомления и т.п.) на **Firebase**.

## Стек
- React 18, TypeScript
- Firebase (Auth, Firestore, Storage, optional Analytics)
- TailwindCSS + styled-components
- react-router-dom, framer-motion

## Быстрый старт

### 1) Установка
```bash
npm install
```

### 2) Переменные окружения
Скопируй пример и заполни своими значениями из Firebase Console:

```bash
cp .env.example .env
```

### 3) Запуск
```bash
npm start
```
Открой `http://localhost:3000`.
